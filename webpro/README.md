# 🏥 Django Project

A full-stack Django web application for user registration, authentication, and basic management system.

---

## 🚀 Features

- User Registration System
- Login & Logout
- Admin Panel
- Static Pages (Home, About, Contact, Gallery)
- Media Upload Support
- SQLite Database

---

## 🛠️ Tech Stack

- Python
- Django
- HTML, CSS, Bootstrap
- SQLite

---

## 🗄️ Database

- File: `db.sqlite3`
